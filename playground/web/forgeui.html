ForgeUI is the SwiftUI-first design system in the iOS Forge Kit — a token-first vocabulary, ~50 native components across atoms, molecules, and organisms, and a complete UIKit ↔ SwiftUI bridge. Semantic colors are dynamic, so dark mode comes for free.

⇄ The seam

UIKit ↔ SwiftUI bridge

Host a SwiftUI view in UIKit, or wrap a UIKit component for SwiftUI — every component crosses the seam.

Inline
Whole screen
SwiftUI → UIKit
UIHostingView<Content>
Embed a SwiftUI view inside a UIKit layout — a cell, header, toolbar slot. No child view controller.
FUIHostingController<Content>
Host a SwiftUI view as a pushable / presentable screen from a Coordinator.
UIKit → SwiftUI
FUIViewRepresentable<V>
Generic escape hatch — wrap any UIView with make + update closures.
FUI*Representable
A named wrapper beside every UIKit component, wiring its callbacks (onTap, onBack, …).
